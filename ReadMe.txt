Usage Instrucutions:
1.) Fill your details like TNo, Password, Date of birth, time of birth etc. If you dont wish to fill that info add ";" at start of the line
2.) Once you filled the ini file, you can run the exe by calling the exe like this : TeluguMatrimonyScrapper.exe TeluguMatrimony.ini > ScrapeResults.tsv
3.) this will write the results into a file "ScrapeResults.tsv" in the same folder. You can open the tsv file in Excel for easy visualization and manipulation
4.) To run the scrapper in a "Fill Missing" mode: TeluguMatrimonyScrapper.exe -FillMissing TeluguMatrimony.ini ScrapeResults.tsv > ScrapeResults_new.tsv
